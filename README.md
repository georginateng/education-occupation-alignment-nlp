# Education-Occupation Alignment using NLP

This project investigates the alignment between university curricula and occupational skill requirements in Korea.

## Data Sources

- KERIS University Curriculum Dataset (2022)
- Career.go.kr Occupational Information API

## Methodology

- Curriculum corpus construction
- Occupation corpus construction
- Sentence-BERT embeddings
- Cosine similarity analysis

## Current Status

- Department mapping completed
- Curriculum data preparation completed
- Occupation data collection pending API approval
